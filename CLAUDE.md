## Project
Faheem Live is a real-time bilingual tutoring MVP for the Gemini Live Agent Challenge.

## Stack
- Next.js frontend
- FastAPI backend
- Firestore
- Cloud Run

## Rules
- Keep changes minimal and modular
- Do not overengineer
- Prioritize working vertical slices
- Preserve existing folder structure

## Commands
- Frontend: npm run dev
- Backend: uvicorn app.main:app --reload
